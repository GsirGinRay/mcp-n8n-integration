# n8n-MCP Integration

## Quick Commands

### Start n8n-MCP Connection
```bash
# 推薦使用自定義斜線命令
/n8n-mcp

# 或使用傳統 npm 方式
npm run n8n-mcp
```

This command launches the n8n-MCP integration and connects Claude Code to your n8n platform.

## Custom Slash Commands

Available slash commands in Claude Code:
- `/n8n-mcp` - 一鍵啟動 n8n-MCP 連接器
